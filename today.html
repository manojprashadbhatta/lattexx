First of all, we should know about LaTeX.

LaTeX (pronounced Lay-tech or Lah-tech) is a high-quality typesetting system used for creating well-structured and professionally formatted documents, especially in academic, scientific, and technical fields. Unlike traditional word processors, LaTeX is not WYSIWYG (What You See Is What You Get); instead, you write plain text with markup commands to define the structure (like sections, formulas, tables, and references). It excels at handling complex mathematical equations, citations, and consistent formatting. LaTeX is widely used for writing research papers, theses, books, and presentations, offering precise control over document layout and style.

Document Setup

\documentclass{article}
Defines the type of document (e.g., article, book).

Packages

\usepackage{amsmath}
Enhances math formatting and structure.
\usepackage{graphicx}
Allows you to include images using \includegraphics.

Start and End Document

\begin{document}
...
\end{document}
All visible content must be placed between these lines.

Text Formatting

\textbf{Bold Text}
\textit{Italic Text}
\underline{Underlined Text}
Formats text as bold, italic, or underlined.

Sections

\section{Main Section}
\subsection{Subsection}
\subsubsection{Subsubsection}
Organizes content into hierarchical headings.

Lists

\begin{itemize}
  \item First item
  \item Second item
\end{itemize}
Creates a bullet list.
\begin{enumerate}
  \item First item
  \item Second item
\end{enumerate}
Creates a numbered list.

Math Mode

Inline: $E = mc^2$
Display:
\[
  a^2 + b^2 = c^2
\]
Allows writing mathematical expressions inline or in display mode.

Line Breaks and Paragraphs

This is a line.\\
(Leave a blank line to start a new paragraph.)
Use \\ for line breaks and blank lines for new paragraphs.

Comments

% This is a comment
Anything after % is ignored by LaTeX.
